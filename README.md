# 🥕와플마켓🧇

- 온라인 중고거래 플랫폼인 당근마켓 앱을 웹 버전으로 클론코딩 해보았습니다.
- 당근마켓의 핵심 기능들인 물품 거래, 커뮤니티 활동, 채팅 등을 중점으로 구현하면서, 새로운 기능인 **경매** 기능을 추가하였습니다.
- 배포 URL: https://toykarrot.shop/
